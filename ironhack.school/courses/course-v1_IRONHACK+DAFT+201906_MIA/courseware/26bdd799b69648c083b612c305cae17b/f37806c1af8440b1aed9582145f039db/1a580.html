Many times, this can be accomplished with a myriad of summary statistics. However, sometimes we find that "a picture is worth a thousand words" and that data visualizations can capture a large amount of data all at once in a clear and concise manner.</p> <h2>Box Plot (aka Box whisker plot)</h2> <p>Box plots are an amazing way to show the distribution of quantitative data and make comparisons across levels of a categorical variables. They help to visualize the 5-number summary as well as create a boundary that will help us distinguish which outliers are extreme. Those 5 numbers are minimum, 25th percentile, 50th percentile (Median), 75th percentile, maximum</p> <p>The box is defined as containing the data between the 25th and 75th percentiles. A line is drawn in the box to show where the median (or 50th percentile) lies. Box plots also have two lines extending from each end of the box called whiskers. The whiskers are typically drawn by finding the smallest and largest points still within 1.5 times the interquartile range minus the lower quartile or 1.5 times the interquartile range plus the upper quartile.</p> <p>The interquartile range is the difference between the first and third quartiles.&nbsp;Since this verbal explanation might be confusing, here are some formulas:</p> <pre>IQR = Third Quartile - First Quartile</pre> <pre>Upper Whisker Limit &lt; Third Quartile + 1.5 * IQR</pre> <pre>Lower Whisker Limit &gt; First Quartile - 1.5 * IQR</pre> <p>With all of this in mind, let's plot a box plot of our vehicles data. We first load the <a href="https://s3-eu-west-1.amazonaws.com/ih-materials/uploads/data-static/data/module-1/vehicles.zip"><span style="-webkit-font-kerning: none; color: #1d8ad0;">vehicles dataset</span></a> using pandas. And then we will plot a box plot of the combined MPG</p> <pre><code>import pandas as pd<br />data = pd.read_csv('~path/vehicles.csv')<br />data.boxplot(column="Combined MPG")</code></pre> <p><img height="246" width="322" src="/assets/courseware/v1/81f2ca2830a59fff5393d0162a7635bc/asset-v1:IRONHACK+DAFT+201906_MIA+type@asset+block/Screen_Shot_2019-05-20_at_9.34.51_AM.png" alt="" /></p> <p>Note that we did not use the <code>.plot</code> function here.We can infer from this box plot that the data is skewed due to a large number of extreme outliers that have very good MPG. We can also see that the median is not exactly in the middle of the box.</p> <h2>Box plots using Seaborn</h2> <p>More detailed plots can be made using the seaborn library.
